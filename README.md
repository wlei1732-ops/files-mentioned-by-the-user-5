# 王磊个人作品集网站

React + Vite 基础版本，内容根据 `王磊个人简历(5).docx` 整理。

## 运行

当前环境没有可用的 `npm` / `pnpm` / `yarn`，因此依赖还没有安装。安装 Node.js/npm 后，在本目录执行：

```bash
npm install
npm run dev
```

然后打开终端显示的本地地址预览。

## 页面结构

- 全屏首页 Hero：视频背景节点、主标题、导航栏、联系按钮
- 个人经历：头像、介绍、联系方式、项目数据
- 精选项目：大卡片展示项目视觉与说明
- 个人优势：能力卡片
- 底部联系方式：整屏收尾页

后续可以直接替换 `public/assets` 中的素材，或在 `src/App.jsx` 里更新项目数据。
